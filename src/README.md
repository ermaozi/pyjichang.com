- 机场运营 2 年及以上。
- 官网能在中国大陆访问，且支持支付宝或微信等国内常用的支付方式。
- 有较为活跃的用户群。
- 存在有效的工单处理通道。
- 价格不要太贵。
- 不要经常掉线，或是订阅链接经常自动更换。
- 网上没有大量的黑料或是纠纷。
